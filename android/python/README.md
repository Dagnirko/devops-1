Python setup

- Install SL4A r6 apk
- Install PythonForAndroid r5 apk
- In Terminal IDE
  - `makedir ~/bin`
  - `cp ./py ~/bin`
  - `echo "export PATH=$PATH:~/bin" >> ~/.bashrc`
  - `py` to enter python

Sgit

- To easily navigate tp git repos cloned using Sgit, run   
  - `echo 'export repo=/mnt/sdcard/Android/data/me.sheimi.sgit/files/repo/' >> ~/.bashrc`
